## Key Metrics (Random Forest v1 - Production Model)

### Test Set Performance (2015-2020)

- RMSE: 2.784°C  [95% CI: 2.713 - 2.853°C]
- MAE:  2.208°C  [95% CI: 2.148 - 2.267°C]
- R²:   0.759    [95% CI: 0.747 - 0.771]

### Monitoring Thresholds

| Metric | Alert | Critical | Action |
|--------|-------|----------|--------|
| RMSE | > 3.18°C | > 3.48°C | Investigate / Retrain |
| MAE | > 2.51°C | > 2.81°C | Investigate / Retrain |
| R² | < 0.70 | < 0.65 | Retrain urgently |

### Performance Breakdown

**By Season:**
- Best: Autumn (RMSE = 2.627°C)
- Worst: Winter (RMSE = 2.894°C)
- Variability: 10.2%

**By Temperature Range:**
- Best: 5-15°C (RMSE = 2.472°C) - 56% of dataset
- Worst: > 25°C (RMSE = 6.410°C) - 0.8% of dataset
